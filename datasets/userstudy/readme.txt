These files contain the detailed data that specify the user study, as well as data collected from participants, and final data analysis:

os2.log - log file of the process that created the experimental tasks
os_pt_BR.csv - specification of the experimental tasks to be uploaded to the Marjory website
os2_pt_BR_responses.csv - experiment data downloaded from the Marjory website after the experiment was closed (include participant's responses)
os2_responses.log - log file of the process that reads the participant's reponses and performs data analysis.
os2_summary_bootstrap.csv - results of the data analysis
userstudy_results.xlsx - an Excel worksheet with the previous csv files imported
